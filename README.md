## Requirements:
* Install Python v3.9+
* Install the following packages:
  * numpy
  * pandas
  * matplotlib
  * seaborn
  * scikit-learn
  * imblearn

## How to Run:
* Using Jupyter
  * Navigate in Terminal/Command Prompt to the directory where you extracted the zip file.
  * Load the jupyter notebook, `ComprehensiveCode.ipynb`, in an environment with the required packages, from above.
  * Run all cells.

## Contents:
* `ComprehensiveCode.ipynb` - the notebook which contains our analysis
* `README.md` - this file
* `census-income.data.csv` - the training dataset
* `census-income.test.csv` - the test dataset
* `census-income.names.txt` - a text file containing information on features

## Outputs:
* A variety of visualizations pertaining to EDA.
* Test results for both cross-validation and test set model evaluation.
